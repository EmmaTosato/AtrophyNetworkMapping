# Brain Journal Style Guide üß†

This document contains the *strict* style rules for the journal *Brain*. **All output text for the paper must rigorously adhere to these guidelines.**

## 1. üÜé Abbreviations
*   **General Rule**: Avoid abbreviations unless very well known (e.g., DNA) or for gene/protein symbols.
    *   Limit to as few as possible, only for repeatedly used words.
    *   **Definition**: Spell out at first mention + (Abbrev). Use abbreviation thereafter.
    *   **Legends**: Define **ALL** abbreviations in every Figure/Table legend, even if defined in text.
*   **Allowed in Title/Abstract** (Commonly Accepted):
    *   AIDS, DNA, mRNA, cDNA, RNA, tRNA
    *   ANOVA
    *   ATP
    *   CNS, CSF
    *   CT, MRI, PET
    *   ECG, EEG, EMG
    *   ELISA, PCR
    *   GABA
    *   GP (doctor)
    *   HIV
    *   PBS
    *   REM

## 2. ‚úçÔ∏è Spelling & Grammar
*   **English Variant**: **Oxford English Spelling**.
    *   Use **-ize** suffixes (e.g., *organize*, *summarize*, *characterize*).
    *   **Exception**: *analyse*, *paralyse*.
    *   Use *behaviour*, *colour*, *signalling*.
*   **Clauses**:
    *   **Restrictive**: Use **'that'** (essential info, no comma). *The book that has a red cover is mine.*
    *   **Non-restrictive**: Use **'which'** (extra info, with comma). *The book, which has a red cover, is mine.*
*   **Capitalization**:
    *   Capitalize after a colon.
    *   Capitalize classifications + number: *Experiment 1*, *Patient 5*, *Day 2*, *Case 3*.
*   **Terminology**:
    *   Use **Male/Female** (not Men/Women).
    *   Use **"Most"** (avoid *"the vast majority"*).
    *   Reserve **"Parameters"** only for scientific measurements.
    *   Avoid **"and/or"**. Use 'and', 'or', or 'either or both'.
    *   **Drugs**: Use **generic names**. If proprietary brand used, include brand in parentheses once in Methods. No supplier location needed.

## 3. üî¢ Numbers & Units
*   **Numbers < 10**: **Spell out** (one, two, ... nine). *Includes ordinals (first, ninth).*
*   **Numbers ‚â• 10**: Use **Arabic numerals** (10, 11, ...).
*   **Exceptions (Always Arabic)**:
    *   Units of measure (e.g., *5 mg*, *3 years*).
    *   Designators (*Case 5*, *Day 2*).
    *   Scores, Confidence Intervals.
    *   Fractions < 1 (spell out? Context implies spell out if general, but usually decimals preferred in science). *Guide says "fractions less than one" spelled out.*
*   **Sentence Start**: **Always spell out** numbers at the start of a sentence.
*   **Dates**: Day Month Year (e.g., *1 January 2006*).
*   **Ranges**: Use **en-dash (‚Äì)** closed up (no spaces).
    *   *pp. 23‚Äì36*, *1939‚Äì45*.
    *   **Repeat symbols** if closed up: *4%‚Äì10%* (not 4‚Äì10%), otherwise *2‚Äì3 mg*.
*   **Confidence Limits (95%)**: Use comma to separate values: *(15, 28)*, **not** *(15-28)*.

## 4. üõë Punctuation
*   **Oxford Comma**: **DO NOT USE** (e.g., write "red, white and blue", NOT "red, white, and blue").
*   **En-dash (‚Äì)** (longer than hyphen `-`):
    *   **Ranges**: *4%‚Äì10%*.
    *   **Connections/Relations**: *Dover‚ÄìCalais crossing*, *AD‚ÄìPSP comparison*, *Mann‚ÄìWhitney U-test*, *editor‚Äìauthor relationship*.
    *   **Parentheticals**: *he was‚Äîso he said‚Äîa businessman* (closed up).
*   **Em-dash (‚Äî)**: Use to introduce a phrase or replace a colon (less formal).
*   **Abbreviations**:
    *   **e.g.** and **i.e.** : **NO comma** after them.
*   **Respectively**: Preceded by a comma (*...values were X and Y, respectively*).
*   **Parentheses**: Order is `{[()]}`. e.g. *{[(...)]}*.

## 5. üìä Statistics & Significance
*   **"Significant"**:
    *   **NEVER** write "statistically significant".
    *   **NEVER** use both a P-value and "significant" for the same point.
    *   Reserve "significant" **only** for statistical testing.
*   **P-values**:
    *   Format: **Upper case P**, **Italics**, **Romand operators**.
    *   Example: ***P* < 0.001**, ***P* = 0.04**.
    *   Asterisks in tables: \* *P* ‚â§ 0.05, \*\* *P* ‚â§ 0.01, \*\*\* *P* ‚â§ 0.001.
*   **Correlations**: Do not say "positive/negative" or "inverse". Describe the observation (e.g., *"an increase in X was associated with..."*).
*   **Equations**:
    *   Simple equations in word format. Complex in MathType.
    *   **Multiplication**: Use `√ó` (not *). e.g. *6 √ó 10^5* (not 6E5).
    *   **F-values**: *F*(2,35) = ...
    *   **t-test**: *t* (lower case, italic).
    *   **U-test**: *U* (Roman U, e.g. *Mann‚ÄìWhitney U-test*).

## 6. üìù Formatting & Citations
*   **Italics**:
    *   **NO** italics for emphasis (use single quotes `‚Äò...‚Äô`).
    *   **Use for**:
        *   Book/Periodical titles.
        *   Latin phrases (*in situ*, *et al.*, *per se*).
        *   Species names (*E. coli*).
        *   Variables (*P*, *t*, *n*).
    *   **Do NOT use for**: Software names, Restriction enzymes.
*   **Quotes**:
    *   Use **single quotes** (`‚Äòtext‚Äô`). Double quotes only for quotes within quotes.
    *   $> 5$ lines: Separate paragraph, indented, smaller font.
*   **Lists**:
    *   Run-in format: *(i) item; (ii) item; and (iii) item*.
*   **Citations (Figures)**:
    *   **Format**: *Fig. 1*, *Figs 3‚Äì6*, *Fig. 2A and B*, *Fig. 3A‚ÄìC*, *Fig. 4A(i and ii) and B*.
    *   **Supplementary**: Cite *after* main text figures.
        *   *Fig. 2A and B and Supplementary material*
        *   *Figs 5 and 6, Supplementary Fig. 3 and Supplementary Tables 1‚Äì7*
    *   **Avoid** repetition in the same paragraph.
*   **Author Names in Text**:
    *   **Avoid** unless historical (Darwin, Newton).
    *   **Incorrect**: *Smith et al. reported...*
    *   **Correct**: *Similar findings have been reported [citation #]*
    *   **Reviews**: *For reviews, see Al-Chalabi et al.^4* (generalized statement allowed for reviews).
